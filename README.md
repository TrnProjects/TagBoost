## üõ† Installation Instructions

### Chrome:
1. Open Chrome `chrome://extensions/`
2. Enable **Developer Mode**.
3. Click **Load Unpacked**.
4. Select the `TagBoost_Main` folder.

### Edge:
1. Open Edge `edge://extensions/`
2. Enable **Developer Mode**.
3. Click **Load Unpacked**.
4. Select the `TagBoost_Main/TagBoost_Edge` folder.
